📐 1. HTML structure

Every document starts with <!DOCTYPE html>, <html>, <head> and <body>.

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width">
  <title>My page</title>
</head>
<body>
  <h1>Hello</h1>
</body>
</html>

<meta> tags for charset, description, viewport, etc. are placed inside <head>.
